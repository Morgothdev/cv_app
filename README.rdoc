== CV APP

Aplikacja ta powstała jako aplikacja z moim cv, między innymi po to, 
by upublicznić miejsce do sprawdzenia aktualnych informacji na mój temat, 
jak również jako przykładowa aplikacja w Ruby on Rails jako jeden z realizowanych projektów.

Aplikacja w tym momencie jest na etapie wczesnego szkieletu. Jednakże w miarę czasu wonego bedzie rozwijana.

Powstaje z użyciem:
  rcpec + guard + spork -> narzędzia do testowania oraz automatyzacji testów
  bootstrap -> CSS
  postgresql
